# appspawn应用孵化器组件

## 简介

应用孵化器，负责接受应用程序框架的命令孵化应用进程，设置其对应权限，并调用应用程序框架的入口。

## 目录

```
base/startup/appspawn_standard
├── include                    # 应用孵化器组件头文件
├── parameter                  # 应用孵化器组件系统参数
├── src                        # 应用孵化器组件源文件
└── test                       # 应用孵化器组件测试代码
```
